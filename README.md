# ListeningTrends

Project is to organize and analyze my favorite tracks better than spotify using somewhat quantifiable audio features such as 'danceability',
   'energy',
   'key',
   'loudness',
   'mode,
   'speechiness',
   'acousticness',
   'instrumentalness',
   'liveness',
   'valence',
   'tempo'.
   All the metrics are provided by Spotify API.
